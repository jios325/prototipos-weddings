# O Weddings by Oasis — Prototipos SEO Editorial

> **12 paginas editoriales** diseñadas para posicionar organicamente en Google, sin depender de paquetes. Enfocadas en destino, experiencia y conocimiento.

**Live site:** [jios325.github.io/prototipos-weddings](https://jios325.github.io/prototipos-weddings/)

## Estrategia

Inspiradas en como **Aman, Belmond Maroma y Four Seasons Naviva** venden bodas de destino sin mencionar paquetes — enfocandose en editorial, destino y experiencia. Cada pagina ataca un keyword cluster de alto volumen para capturar trafico organico y guiar al usuario hacia la conversion.

### Funnel de Conversion

```
TOP FUNNEL (alto volumen, descubrimiento)
  Couples Guide → Honeymoon → descubren O Weddings

MEDIO FUNNEL (consideracion)
  Destination Wedding → Beach Wedding → Venues → All-Inclusive

BAJO FUNNEL (alta intencion de compra)
  Cost Guide → Legal Guide → Mayan Ceremony → Elopement

CONVERSION
  → Landing de paquetes → O Ceremony / O For You / O Romance / O Forever
```

## Contexto Competitivo

Analisis basado en investigacion de Gemini Deep Research + analisis complementario de 11 marcas hoteleras:

| Marca | Modelo | Diseño | Conversion |
|-------|--------|--------|------------|
| Aman | Consultivo (sin paquetes) | ★★★★★ | ★★★★★ |
| Four Seasons Naviva | Paquetes transparentes | ★★★★★ | ★★★★★ |
| Viceroy Riviera Maya | Hibrido editorial + paquetes | ★★★★ | ★★★★★ |
| Belmond Maroma | Consultivo editorial | ★★★★★ | ★★★★ |
| Hotel Xcaret | Paquetes + destino | ★★★★ | ★★★★ |

**O Weddings adopta el modelo hibrido** (Viceroy): paginas editoriales SEO para captar trafico + paginas de paquetes para convertir.
